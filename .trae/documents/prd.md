## 1. 产品概述

城市公厕保洁巡检系统——面向市政环卫场景的公厕清洁巡检管理平台。保洁员在地图/列表中维护巡检点位，片区主管实时查看清洁状态与异常，市民监督员仅可查看公示信息。异味等级高的点位自动置顶，耗材不足显示补给按钮，关闭点位禁止提交巡检。

- 目标用户：市政环卫保洁员、片区主管、市民监督员
- 核心价值：提升公厕巡检效率、异常快速响应、公众透明监督

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 保洁员 | 管理员分配 | 维护点位、提交巡检、标记耗材、查看自己负责的点位 |
| 片区主管 | 管理员分配 | 查看片区所有点位状态、审核巡检、处理异常、导出报表 |
| 市民监督员 | 自由注册 | 仅查看公示信息（开放状态、清洁等级、异味等级） |

### 2.2 功能模块

1. **巡检首页**：地图总览 + 点位列表，异味置顶，异常高亮
2. **巡检录入页**：填写巡检表单，关闭点位禁止提交
3. **点位管理页**：筛选、搜索、查看详情、耗材补给
4. **数据导出页**：筛选条件导出 CSV

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 巡检首页 | 地图总览 | 在地图上展示所有公厕点位，不同颜色标记异味等级，高亮异常点位 |
| 巡检首页 | 点位列表 | 列表模式切换，异味等级高的置顶，异常点位红色标记，关闭点位灰色显示 |
| 巡检首页 | 角色切换 | 顶部角色选择器，切换保洁员/主管/监督员视图 |
| 巡检录入页 | 巡检表单 | 清洁状态、异味等级、耗材情况、照片上传、备注，关闭点位禁止提交 |
| 巡检录入页 | 耗材补给 | 耗材不足时显示补给按钮，点击标记补给请求 |
| 点位管理页 | 筛选面板 | 按片区、异味等级、清洁状态、开放状态筛选 |
| 点位管理页 | 点位详情 | 点位基础信息、历史巡检记录、耗材库存 |
| 数据导出页 | 导出配置 | 选择筛选条件、时间范围，导出 CSV |

## 3. 核心流程

**巡检录入流程**：保洁员选择点位 → 判断是否开放 → 开放则填写巡检表单 → 提交巡检记录 → 异常自动高亮通知主管

```mermaid
flowchart TD
    "保洁员登录" --> "选择点位"
    "选择点位" --> "点位是否开放?"
    "点位是否开放?" -- 否 --> "提示: 关闭点位禁止巡检"
    "点位是否开放?" -- 是 --> "填写巡检表单"
    "填写巡检表单" --> "异味/耗材是否异常?"
    "异味/耗材是否异常?" -- 是 --> "标记异常+通知主管"
    "异味/耗材是否异常?" -- 否 --> "提交成功"
    "标记异常+通知主管" --> "提交成功"
```

**主管监控流程**：主管登录 → 查看片区概览 → 发现异常点位 → 处理异常 → 审核巡检

```mermaid
flowchart TD
    "主管登录" --> "查看片区概览"
    "查看片区概览" --> "发现异常点位"
    "发现异常点位" --> "处理异常"
    "处理异常" --> "审核巡检记录"
    "审核巡检记录" --> "导出报表"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青色 `#0F766E`（环卫/市政感）+ 琥珀色 `#D97706`（警示/异常）
- 辅助色：石板灰 `#475569`（中性信息）、红色 `#DC2626`（严重异常）、翡翠绿 `#059669`（正常状态）
- 按钮风格：圆角 8px，轻微阴影，hover 时上浮
- 字体：Noto Sans SC（中文场景）+ DM Sans（数字/英文）
- 布局风格：左侧导航 + 右侧内容区，卡片式模块，顶部状态条
- 图标风格：Lucide 线性图标

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 巡检首页 | 地图总览 | 全幅地图，圆形标记点，红/橙/绿三色异味等级，脉冲动画异常标记 |
| 巡检首页 | 点位列表 | 卡片列表，置顶排序，灰色半透明覆盖关闭点位，红色边框异常点位 |
| 巡检首页 | 角色切换 | 顶部胶囊选择器，深青色激活态 |
| 巡检录入页 | 巡检表单 | 分组表单，星级评分组件，图片上传区域，底部固定提交按钮 |
| 巡检录入页 | 耗材补给 | 黄色闪烁补给按钮，点击后变为已请求状态 |
| 点位管理页 | 筛选面板 | 顶部多条件筛选栏，下拉选择+标签式筛选 |
| 数据导出页 | 导出配置 | 日期范围选择器，多选筛选条件，大号导出按钮 |

### 4.3 响应式设计

桌面优先设计，平板适配侧边栏折叠，移动端适配单列布局和底部导航

### 4.4 异常高亮规则

- 异味等级 ≥ 4：红色脉冲边框 + 置顶
- 异味等级 = 3：橙色边框
- 耗材不足：黄色闪烁补给按钮
- 点位关闭：灰色半透明覆盖 + 禁止提交提示
